# Shift_Jump-The_Glitches
Unity motoru ile geliştirilmiş; farklı dünyalar, temalar ve zorluk seviyeleri içeren dinamik bir 2D parkur oyunudur. Hitit Oyun Kulübü bünyesindeki 'The Glitches' geliştirici ekibi tarafından hazırlanan bu proje, akıcı oynanış mekanikleri ve seviye tabanlı ilerlemeye odaklanır.
